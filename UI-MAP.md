## 名称速查表

| 编号 | 名称 | 角色 | 关键视觉特征 |
|------|------|------|-------------|
| — | **Shell** | 无交互背景层 | 暖卡其渐变（light）/ 深色渐变（dark），面板间隙透出 |
| ① | **TitleBar** | 窗口顶栏 | 40px 高，毛玻璃半透明，品牌名 + WS 状态点 + 设置齿轮 |
| ② | **Sidebar** | 会话列表侧栏 | 260px 宽，可折叠，`proma-panel`（圆角+阴影），列表项圆角 10px |
| ③ | **PaperPanel** | 论文阅读面板 | 左侧主面板（40%-80% 宽），内含 TabBar + PaperViewer / NotesPanel |
| ③a | **TabBar** | 论文页签栏 | 38px 高，笔记 Tab + 论文 Tab，选中态底部绿色下划线 |
| ③b | **PaperViewer** | 论文内容区 | PDF/DOCX/TXT 渲染，支持缩放、翻译对照 |
| ③c | **NotesPanel** | 共享笔记 | Textarea，AI 和用户共享读写 |
| ④ | **ConversationPanel** | AI 对话面板 | 右侧面板（20%-60% 宽），`proma-panel` |
| ④a | **ScreenContext** | AI 视野状态条 | 显示 AI 当前"看"到的内容来源和段落 |
| ④b | **MessageList** | 消息列表 | 用户/AI 消息气泡，引用，知识图谱关联 |
| ④c | **QueryInput** | 输入区 | 麦克风按钮 + 文本框（内嵌 Enter 图标）+ 语音提示 |
| ⑤ | **SettingsPanel** | 设置弹窗 | 模态浮层，左侧导航 + 右侧内容，毛玻璃遮罩 |
| — | **Divider** | 面板分隔条 | 5px 宽，中间圆角把手，拖拽调整左右比例 |
